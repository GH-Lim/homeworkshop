## 3 번

### 다음과 같은 이차방정식이 있을 때 근을 찾는 수식을 파이썬 코드를 이용하여 출력해 보시오.

x^2 + 4x - 21

=> ax^2 + bx -21 형태로 생각하고 변수를 설정합니다.

```python
a = 1
b = 4
c = -21

x1 = ((-b) + (b ** 2 - 4 * a * c) ** 0.5) / (2 * a)
x2 = ((-b) - (b ** 2 - 4 * a * c) ** 0.5) / (2 * a)

print(f'근1 : {x1}\n근2 : {x2}')
```
